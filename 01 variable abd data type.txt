Rules for Writing Variables in Python:
Can use:
Letters: A–Z (uppercase), a–z (lowercase)
Numbers: 0–9
Underscore: _

Cannot start with a number ✅ Valid: age1, name_2 ❌ Invalid: 1age, 2name
Cannot use special symbols like:   ❌ %, @, #, !, &, *, etc.
No spaces allowed Use _ instead✅ first_name ❌ first name
Variable names are case-sensitive : Name and name are different
Should not use Python keywords:❌ Examples: if, while, for, class, etc.
Variable name: simple easy and meaning full name used 

Variable Data Type

Data              Type	            Example	Use
int	               10	            Whole numbers
float	           3.14         	Decimal numbers
str               (string)	        "Hello"	Text
bool	          True, False	    True/False values
list              [1, 2, 3]	        Ordered collection of items
tuple	          (1, 2, 3)	        Ordered, unchangeable list
dict	       {"a": 1, "b": 2}	     Key-value pairs (like a map)
set	              {1, 2, 3}      	Unordered unique items
NoneType	       None	No           value
